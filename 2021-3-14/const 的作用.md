#### const 的作用

***

1.	成员函数中的const
>主要是将this指针的类型由Type*const变为const Type* const，防止调用函数对对象的修改。同时允许const对象进行调用。

2.	修饰类型
> const Type A和Type const A：表示常量，值不允许修改。如果没有默认构造函数，需要初始化。

3.	const与指针
> const Type\* p：表示常量指针，p所指向的地址可以修改，但是不允许使用指针p修改当前地址的所存储的值。
> Type\* const P：表示指针常量，p的值不可以修改，即p所指向的地址不可以修改。\*p的值可以修改。

4.	const 修饰函数参数
>用其它对象的数据来初始化当前对象，并没有期望更改其它对象的数据
